# Brainｽﾋﾟｷ / Brain스피키

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　

### 日本語

BrainｽﾋﾟｷはBrainf\*\*kの派生言語です。

名前の通りですし特に多くを語る必要もありませんね。



Brainsupiki\_interpreter.py : インタープリターです。

BF\_to\_Brainsupiki.py : Brainf\*\*kとBrainｽﾋﾟｷの間でコードを変換できます。



#### 命令

半角カタカナ・韓国語　どちらでもOKです。

* `ﾑﾙｺﾞﾙﾚｼﾞ` `물걸레질`　ポインタをインクリメント （右にずらす）
* `ﾎﾊﾞｷﾞ` `호박이`　ポインタをデクリメント (左にずらす）
* `ﾁｮﾜﾖ~` `좋아요~`　	ポインタの値をインクリメント
* `ｽﾝﾊﾞｺｯﾁ` `숨바꼭질`　ポインタの値をデクリメント
* `ｳｱｱ!` `흐으아악!`　ポインタの値を出力
* `ｽﾋﾟｷﾃﾞﾘｼﾞﾊﾞｾﾞﾖ!` `스피키 네르지 마세요!`　入力から1バイト読み込んで、ポインタが指す値に代入
* `ｽﾋﾟｷﾓﾘﾁｬﾊﾞﾀﾞﾝｷﾞｼﾞﾊﾞｾﾞﾖ!` `스피키 머리 잡아당기지 마세요!`	ポインタの指す値が0なら、後の]までジャンプ
* `ｽﾋﾟｷｦｲｼﾞﾒﾇﾝﾃﾞ...` `스피키 열심히 했는데...`　ポインタの指す値が0でなければ、前の\[までジャンプ



### 한국어

Brain스피키는 Brainf\*\*k의 파생 언어입니다.

이름 그대로이며 특별히 더 설명할 필요도 없겠네요.



Brainsupiki\_interpreter.py : 인터프리터입니다.

BF\_to\_Brainsupiki.py : Brainf\*\*k와 Brain스피키 간에 코드를 변환할 수 있습니다.



#### 명령어

반각 가타카나/한국어 모두 사용 가능합니다.

* `ﾑﾙｺﾞﾙﾚｼﾞ` `물걸레질`　포인터를 인크리먼트 (오른쪽으로 이동)
* `ﾎﾊﾞｷﾞ` `호박이`　포인터를 디크리먼트 (왼쪽으로 이동)
* `ﾁｮﾜﾖ~` `좋아요~`　포인터 값을 인크리먼트
* `ｽﾝﾊﾞｺｯﾁ` `숨바꼭질`　포인터 값을 디크리먼트
* `ｳｱｱ!` `흐으아악!`　포인터 값을 출력
* `ｽﾋﾟｷﾃﾞﾘｼﾞﾊﾞｾﾞﾖ!` `스피키 네르지 마세요!`　입력에서 1바이트 읽어서 포인터가 가리키는 값에 대입
* `ｽﾋﾟｷﾓﾘﾁｬﾊﾞﾀﾞﾝｷﾞｼﾞﾊﾞｾﾞﾖ!` `스피키 머리 잡아당기지 마세요!`　포인터가 가리키는 값이 0이면, 뒤의 ]까지 점프
* `ｽﾋﾟｷｦｲｼﾞﾒﾇﾝﾃﾞ...` `스피키 열심히 했는데...`　포인터가 가리키는 값이 0이 아니면, 앞의 \[까지 점프
